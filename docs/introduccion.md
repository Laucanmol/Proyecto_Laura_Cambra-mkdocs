# Introducción

MkDocs es una herramienta que permite crear **documentación web fácilmente** usando Markdown.

## Ventajas

- Fácil de usar
- Rápido de configurar
- Ideal para proyectos de programación

!!! tip "Consejo"
    Usa Markdown para escribir documentación clara y sencilla.

## Lista de tecnologías

1. Python
2. Markdown
3. MkDocs
