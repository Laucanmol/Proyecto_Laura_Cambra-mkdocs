# Uso del proyecto

Después de crear el proyecto puedes iniciar el servidor.

## Ejecutar el servidor

```bash
mkdocs serve
```

Después abre el navegador en:

```
http://127.0.0.1:8000
```

## Crear nuevas páginas

Solo debes crear archivos `.md` dentro de la carpeta `docs`.

!!! success "Correcto"
    Si ves la página en el navegador, tu proyecto funciona correctamente.
