
# Instalación

Para instalar MkDocs necesitas **Python**.

## Instalar MkDocs

```bash
pip install mkdocs
```

## Crear proyecto

```bash
mkdocs new mi_proyecto
```

## Ejecutar servidor

```bash
mkdocs serve
```

!!! warning "Importante"
    Debes tener Python instalado antes de instalar MkDocs.
